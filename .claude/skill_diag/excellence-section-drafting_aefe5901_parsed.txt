Target: \u226540% improvement in plan success rate over LLM-baseline architectures (ReAct, Reflexion) on standardised benchmarks (WebArena, SWE-bench, GAIA) for tasks requiring more than 10 sequential steps.\n\nOBJ-2 \u2014 Adaptive Memory Architecture (Lead: BIIS, Month 30): Implement a tri-store memory architecture integrating working, episodic, and semantic memory with a learned controller governing information flow, retrieval, and forgetting.
